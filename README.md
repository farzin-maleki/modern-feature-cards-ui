# Interactive Feature Cards UI

A modern feature card component built using HTML and CSS.  
This project demonstrates interactive card elements with animated corner borders that expand when hovering over the cards.

## Technologies Used
- HTML5
- CSS3

## Features
- Modern card UI layout
- Hover animation effects
- Expanding colored corner borders
- Icon-based feature cards
- Clean and minimal design

## Purpose of the Project
This project was created as part of my front-end development practice to improve my skills in building interactive UI components and hover animations using CSS.
